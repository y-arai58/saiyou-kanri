# 採用フロー定義

採用管理アプリで管理する6種類のフローを定義したドキュメントです。

> 採用サイト経由のエントリーはフォーム入力もエントリー時に完了するため、フォーム送付・待ちステップはありません。

---

## フロー一覧

| フローID | 表示名 | ステップ数 |
|---|---|---|
| `chuto_casual` | 中途｜カジュアル面談 | 5 |
| `chuto_mensetsu` | 中途｜採用面接 | 7 |
| `shinsotsu_kaisetsu` | 新卒｜会社説明 | 4 |
| `shinsotsu_honsenkou` | 新卒｜本選考 | 11 |
| `intern_site_eng` | 長期インターン｜採用サイト | 7 |
| `intern_zero_eng` | 長期インターン｜ゼロワン | 6 |

---

## 中途採用

### 中途｜カジュアル面談（`chuto_casual`）

```
エントリー受付（フォーム入力済み）
  → 希望日程確認
  → 日程調整   ← 日程入力（scheduledDate）
  → 面談
  → 完了
```

| # | ステップ | ステップID | 担当者アクション |
|---|---|---|---|
| 1 | エントリー受付 | `entry` | Slack chでフォーム入力内容を確認する |
| 2 | 希望日程確認 | `schedule` | メールで希望日程を確認 |
| 3 | 日程調整 | `schedule` | 担当者と日程を決定し、応募者へメールで連絡する |
| 4 | 面談 | `casual` | カジュアル面談を実施する |
| 5 | 完了 | `done` | — |

---

### 中途｜採用面接（`chuto_mensetsu`）

```
エントリー受付（フォーム入力済み）
  → 書類選考
  → 希望日程確認
  → 日程調整   ← 日程入力（interviewDate）
  → 面接
  → 面接合否
  → 採用決定
```

| # | ステップ | ステップID | 担当者アクション |
|---|---|---|---|
| 1 | エントリー受付 | `entry` | Slack chでフォーム入力内容を確認する |
| 2 | 書類選考 | `shorui` | フォーム内容をもとに書類選考する |
| 3 | 希望日程確認 | `shorui_pass` | メールで希望日程を確認 |
| 4 | 日程調整 | `shorui_pass` | 担当者と日程を決定し、応募者へメールで連絡する |
| 5 | 面接 | `interview_scheduled` | 面接を実施する |
| 6 | 面接合否 | `interview` | 合否を判断し、応募者へ結果を連絡する |
| 7 | 採用決定 | `done` | — |

## 新卒採用

### 新卒｜会社説明（`shinsotsu_kaisetsu`）

> フォーム内で日程候補を選択済みのためエントリー時点で候補日が確認できます。

```
エントリー受付（フォーム内で日程候補選択済み）
  → 日程調整   ← 日程入力（scheduledDate）
  → 会社説明
  → 完了
```

| # | ステップ | ステップID | 担当者アクション |
|---|---|---|---|
| 1 | エントリー受付 | `entry` | Slack chでフォーム入力内容・希望日程を確認する |
| 2 | 日程調整 | `schedule` | フォームの希望日程から担当者と日程を決定 |
| 3 | 会社説明 | `kaisetsu` | 会社説明を実施する |
| 4 | 完了 | `done` | — |

---

### 新卒｜本選考（`shinsotsu_honsenkou`）

```
エントリー受付（フォーム入力済み）
  → 書類選考
  → 1次希望日程確認
  → 1次日程調整   ← 日程入力（interview1Date）
  → 1次面接
  → 1次面接合否
  → 2次希望日程確認
  → 2次日程調整   ← 日程入力（interview2Date）
  → 2次面接
  → 2次面接合否
  → 採用決定
```

| # | ステップ | ステップID | 担当者アクション |
|---|---|---|---|
| 1 | エントリー受付 | `entry` | Slack chでフォーム入力内容を確認する |
| 2 | 書類選考 | `shorui` | フォーム内容をもとに書類選考する |
| 3 | 1次希望日程確認 | `shorui_pass` | メールで希望日程を確認 |
| 4 | 1次日程調整 | `shorui_pass` | 担当者と日程を決定し、応募者へメールで連絡する |
| 5 | 1次面接 | `interview1` | 面接を実施する |
| 6 | 1次面接合否 | `interview1_judge` | 1次選考の合否を判断する |
| 7 | 2次希望日程確認 | `interview2_sched` | 代表スケジュールを確認し、候補日5つ送付 |
| 8 | 2次日程調整 | `interview2_sched` | 日程が確定したら、応募者へメールで連絡 → カレンダーへ応募者情報を登録 |
| 9 | 2次面接 | `interview2` | 面接を実施する |
| 10 | 2次面接合否 | `interview2_judge` | 合否を判断し、応募者へ結果を連絡する |
| 11 | 採用決定 | `done` | — |

## 日程フィールドの対応表

| フィールド名 | 用途 | 使用フロー |
|---|---|---|
| `scheduledDate` | 会社説明・面談の日程 | 中途カジュアル面談、新卒会社説明 |
| `interviewDate` | 面接の日程 | 中途採用面接、インターン（採用サイト・ゼロワン） |
| `interview1Date` | 1次面接の日程 | 新卒本選考 |
| `interview2Date` | 2次面接の日程 | 新卒本選考 |
