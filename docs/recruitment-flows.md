# 採用フロー定義

採用管理アプリで管理する6種類のフローを定義したドキュメントです。

> 採用サイト経由のエントリーはフォーム入力もエントリー時に完了するため、フォーム送付・待ちステップはありません。

---

## フロー一覧

| フローID | 表示名 | ステップ数 |
|---|---|---|
| `chuto_casual` | 中途｜カジュアル面談 | 4 |
| `chuto_mensetsu` | 中途｜採用面接 | 6 |
| `shinsotsu_kaisetsu` | 新卒｜会社説明 | 4 |
| `shinsotsu_honsenkou` | 新卒｜本選考 | 9 |
| `intern_site_eng` | 長期インターン｜採用サイト | 6 |
| `intern_zero_eng` | 長期インターン｜ゼロワン | 5 |

---

## 中途採用

### 中途｜カジュアル面談（`chuto_casual`）

```
エントリー受付（フォーム入力済み）
  → 日程調整中   ← 日程入力（scheduledDate）
  → 面談（日程確定済み）
  → 面談完了
```

| # | ステップ | ステップID | 担当者アクション |
|---|---|---|---|
| 1 | エントリー受付 | `entry` | Slack chでフォーム入力内容を確認する |
| 2 | 日程調整中 | `schedule` | メールで希望日程を確認し、担当者と日程を決定。確定日程を応募者へメールで連絡する |
| 3 | 面談（日程確定済み） | `casual` | カジュアル面談を実施する |
| 4 | 面談完了 | `done` | — |

---

### 中途｜採用面接（`chuto_mensetsu`）

```
エントリー受付（フォーム入力済み）
  → 書類選考中
  → 書類通過・日程調整中   ← 日程入力（interviewDate）
  → 面接（日程確定済み）
  → 面接実施済み
  → 採用決定
```

| # | ステップ | ステップID | 担当者アクション |
|---|---|---|---|
| 1 | エントリー受付 | `entry` | Slack chでフォーム入力内容を確認する |
| 2 | 書類選考中 | `shorui` | フォーム内容をもとに書類選考する |
| 3 | 書類通過・日程調整中 | `shorui_pass` | メールで希望日程を確認し、担当者と日程を決定。確定日程を応募者へメールで連絡する |
| 4 | 面接（日程確定済み） | `interview_scheduled` | 面接を実施する |
| 5 | 面接実施済み | `interview` | 合否を判断し、応募者へ結果を連絡する |
| 6 | 採用決定 | `done` | — |

## 新卒採用

### 新卒｜会社説明（`shinsotsu_kaisetsu`）

> フォーム内で日程候補を選択済みのためエントリー時点で候補日が確認できます。

```
エントリー受付（フォーム内で日程候補選択済み）
  → 日程確定・連絡   ← 日程入力（scheduledDate）
  → 会社説明（日程確定済み）
  → 会社説明完了
```

| # | ステップ | ステップID | 担当者アクション |
|---|---|---|---|
| 1 | エントリー受付 | `entry` | Slack chでフォーム入力内容・希望日程を確認する |
| 2 | 日程確定・連絡 | `schedule` | フォームの希望日程から担当者と日程を決定し、確定日程を応募者へメールで連絡する |
| 3 | 会社説明（日程確定済み） | `kaisetsu` | 会社説明を実施する |
| 4 | 会社説明完了 | `done` | — |

---

### 新卒｜本選考（`shinsotsu_honsenkou`）

```
エントリー受付（フォーム入力済み）
  → 書類選考中
  → 書類通過・1次日程調整中   ← 日程入力（interview1Date）
  → 1次面接（日程確定済み）
  → 1次面接実施済み
  → 2次日程調整中              ← 日程入力（interview2Date）
  → 2次面接（日程確定済み）
  → 2次面接実施済み
  → 採用決定
```

| # | ステップ | ステップID | 担当者アクション |
|---|---|---|---|
| 1 | エントリー受付 | `entry` | Slack chでフォーム入力内容を確認する |
| 2 | 書類選考中 | `shorui` | フォーム内容をもとに書類選考する |
| 3 | 書類通過・1次日程調整中 | `shorui_pass` | メールで希望日程を確認し、担当者と日程を決定。確定日程を応募者へメールで連絡する |
| 4 | 1次面接（日程確定済み） | `interview1` | 面接を実施する |
| 5 | 1次面接実施済み | `interview1_judge` | 1次選考の合否を判断する |
| 6 | 2次日程調整中 | `interview2_sched` | 代表スケジュール確認 → 候補日5つ送付 ＋ 応募者情報を代表へ共有。確定日程を応募者へメールで連絡する |
| 7 | 2次面接（日程確定済み） | `interview2` | 面接を実施する |
| 8 | 2次面接実施済み | `interview2_judge` | 合否を判断し、応募者へ結果を連絡する |
| 9 | 採用決定 | `done` | — |

### 長期インターン｜ゼロワン・エンジニア（`intern_zero_eng`）

> ゼロワン経由のため書類選考は通過済みの状態からスタートします。

```
書類選考通過
  → 面接日程調整中   ← 日程入力（interviewDate）
  → 面接（日程確定済み）
  → 面接実施済み
  → 採用決定
```

| # | ステップ | ステップID | 担当者アクション |
|---|---|---|---|
| 1 | 書類選考通過 | `shorui_pass` | 面接の日程を調整する |
| 2 | 面接日程調整中 | `schedule` | 確定次第で担当者へ連絡。確定日程を応募者へメールで連絡する |
| 3 | 面接（日程確定済み） | `interview_scheduled` | 面接を実施する |
| 4 | 面接実施済み | `interview` | 合否を判断し、応募者へ結果を連絡する |
| 5 | 採用決定 | `done` | — |

## 日程フィールドの対応表

| フィールド名 | 用途 | 使用フロー |
|---|---|---|
| `scheduledDate` | 会社説明・面談の日程 | 中途カジュアル面談、新卒会社説明 |
| `interviewDate` | 面接の日程 | 中途採用面接、インターン（採用サイト・ゼロワン） |
| `interview1Date` | 1次面接の日程 | 新卒本選考 |
| `interview2Date` | 2次面接の日程 | 新卒本選考 |
